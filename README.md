## 📖 Sobre o projeto

A Calculadora Estatística é uma aplicação web desenvolvida durante a graduação em **Análise e Desenvolvimento de Sistemas**, em novembro de 2024.

O projeto foi criado com o objetivo de facilitar o cálculo de probabilidades utilizando as distribuições **Binomial** e **de Poisson**, além de apresentar medidas estatísticas importantes de forma prática e intuitiva.

Foi um dos primeiros desafios de programação da graduação e representou um importante passo na aplicação de conceitos matemáticos e desenvolvimento web.
